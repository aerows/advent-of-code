Advent of code 2024 solutions in Python by Daniel Hallin (hallin.daniel@gmail.com).

## Commands

### Setup 
```bash
./setup.ps1
```

```bash
python ./2024/15/solution.py
```

```bash
# Creates copy of template
./make-new.ps1

```
